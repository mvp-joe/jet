# Jet
Jet is a simple command line utility that runs a JS script file against JSON input to product JSON output.

In other words, a convenient extract & transform tool for JSON.

## Usage
```bash
$ jet < <input.json> -f <script.js> 
```

## Example
```bash
$ task basic-test
$ task advanced-test
```
